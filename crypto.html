Ciphers

Caesar Cipher
=

The Caesar cipher is one of the simplest and oldest types of substitution cipher. It shifts each letter of the plaintext by a certain number of positions down the alphabet.

Substitution Cipher

A substitution cipher replaces each letter of the plaintext with another letter. Unlike the Caesar cipher, the substitution can vary and doesn't have to be a fixed shift.

Vigenère Cipher

The Vigenère cipher is a more complex form of substitution cipher that uses a keyword to determine the shifts for each letter in the plaintext.

Playfair Cipher

The Playfair cipher encrypts digraphs (pairs of letters). It uses a 5x5 matrix filled with letters of the alphabet (usually excluding 'J' and merging it with 'I').

Monoalphabetic Cipher

This is a substitution cipher where each letter of the alphabet is mapped to a unique other letter. This mapping remains the same for the entire message.

Affine Cipher

The Affine cipher is a type of substitution cipher that uses mathematical functions to map each letter to another letter. It uses two keys: a and b.

One-Time Pad

The one-time pad (OTP) is a theoretically unbreakable cipher that uses a random key that is as long as the message itself. The key is used only once and then discarded.
